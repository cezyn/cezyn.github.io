使用 Sublime Text 写 Markdown

默认 Sublime Text 是不支持 Markdown 语法高亮和预览的，对于万能的 Sublime Text，这点事情一定有插件能解决。

Sublime Text 3 安装Package Control，挺不能理解为什么不默认含在 app 里
快捷键**ctrl+`**或者 View > Show Console 菜单打开控制台

安好重启后，用万能快捷键Ctrl+Shift+P，调出菜单Package Control Install，再输入Markdown关键字，就能发现很多 Markdown 插件。
经过反复尝试，还是一个叫Markdown Editing的比较好使，直接使编辑器在编辑时所见即所得，只是这个默认灰色的颜色…
